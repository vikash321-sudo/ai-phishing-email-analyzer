# 🛡️ AI-Powered Phishing Email Analyzer

An AI-powered cybersecurity tool that analyzes suspicious emails and detects phishing indicators using OpenAI API.

## 🚀 Features

- Detects phishing indicators
- Identifies suspicious links
- Analyzes urgency & social engineering tactics
- AI-generated risk analysis
- Simple Flask web interface
- Dark cybersecurity-themed UI

## 🛠️ Tech Stack

- Python
- Flask
- OpenAI API
- HTML/CSS
